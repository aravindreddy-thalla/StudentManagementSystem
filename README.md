#  Student Management System
A professional Python-based Student Management System built using Object-Oriented Programming principles and file handling. 
This application allows users to manage student records efficiently through a command-line interface with persistent JSON storage.


##  Project Overview
The Student Management System is designed to perform CRUD operations (Create, Read, Update, Delete) on student records. It demonstrates clean code structure, modular design, exception handling, and data persistence using JSON.
This project reflects practical software development practices suitable for real-world backend applications and showcases strong fundamentals in Python development.


##  Features
-  Add new student records
-  View all students
-  Update existing student details
-  Delete student records
-  Persistent data storage using JSON
-  Custom exception handling
-  Modular and scalable code structure
-  Object-Oriented Programming implementation

##  Tech Stack
- **Language: Python 3
- **Concepts: OOP, Exception Handling, File Handling
- **Storage: JSON
- **Version Control:  Git
- **Repository Hosting:  GitHub
